Logins API Test File

This file allows the browser.experiments.logins.* API methods to be called by programmatically clicking buttons. The results of the API invocation can then be verified in the parent process, by calling LoginManager methods. The return values can be checked by looking at the results logged in the named <pre> fields.

Click the Add button to add the test login. 




Click the Get button to get the login with the test ID.
  
  



Click the Get All button to get all logins.
  
  



Click the Remove button to remove the test login.
  
  



Click the Update button to modify the test login, setting the username to 'updated'.
  
  



Click the Touch button to modify the test login, updating its timeLastUsed and
   incrementing its timesUsed counter.
  
  



Click the Register Listeners button to register listeners for the onAdded,
   onUpdated, and onRemoved events. Any events will be appended to a list in
   the results field. Click the 'Clear results' button to clear the previous
   results.
  
  
  



Click the Enable / Disable buttons to change login saving permission for
   this extension

  
  
  



Click the Register / Unregister buttons to change the management URI
  preference fro this extension
